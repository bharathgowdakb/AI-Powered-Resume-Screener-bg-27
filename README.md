# 🧠 AI-Powered Resume Screener

This project uses NLP and Machine Learning to automatically analyze and rank resumes based on a job description.

## 🚀 Features
- Upload PDF resumes
- Match with job descriptions
- Skill/keyword extraction
- Match score display

## 📦 Built With
- Python
- Streamlit
- scikit-learn
- NLTK
- PyPDF2
